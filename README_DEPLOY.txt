Recommended automatic deploy: Netlify

1. Push this folder to a GitHub repository.
2. In Netlify, choose Add new site -> Import from Git.
3. Select the repository.
4. Build command: leave empty.
5. Publish directory: .
6. Deploy.

This folder already includes:
- netlify.toml
- 404.html
- robots.txt
- sitemap.xml
- canonical / Open Graph / Twitter meta tags

Optional: connect your custom domain after the first deploy.
